NAME
    SwiftWin.ps1

SYNOPSIS
    Script for swiftly completing various Windows maintenance tasks.

SYNTAX
    SwiftWin.ps1 [-?] [-Verbose] [-Reset]

DESCRIPTION
    The `SwiftWin.ps1` script runs through a series of Windows maintenance
    tasks I commonly perform when cleaning up and optimizing a system.
